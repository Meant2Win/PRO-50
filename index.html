PHP

A web server programming language

PHP 7 is the latest stable release. It is a a powerful tool for making dynamic and interactive Web pages. PHP is a widely-used, free, and efficient alternative to competitors such as Microsoft's ASP.

PHP Example:

                    <!DOCTYPE html>
                        <html>
                        <body>
                        
                        <?php
                        $color = "red";
                        $COLOR = "green";
                        $coLOR = "blue";
                        echo "My car is " . $color . "<br>";
                        echo "My house is " . $COLOR . "<br>";
                        echo "My boat is " . $coLOR . "<br>";
                        ?>
                        
                        </body>
                        </html>

CSS

The language for styling web pages

CSS stands for Cascading Style Sheets. It is a style sheet language which is used to describe the look and formatting of a document written in markup language. It is generally used with HTML to change the style of web pages and user interfaces.

CSS Example:

                    * {
                        padding: 0;
                        margin: 0;
                        box-sizing: border-box;
                        text-decoration: none;
                        list-style-type: none;
                      }

                      body {
                        background-color: lightblue;
                      }
                      
                      h1 {
                        color: white;
                        text-align: center;
                      }
                      
                      p {
                        font-family: verdana;
                      }

HTML

The language for building web pages

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements.

HTML Example:

                    <!DOCTYPE html>
                        <html lang="en">
                        <head>
                          <title>HTML Example</title>
                          <meta charset="utf-8">
                          <meta name="viewport" content="width=device-width, initial-scale=1">
                        </head>
                        <body>
                          
                        <div class="container">
                          <h1>My First HTML Page</h1>
                          <p>This part is inside a .container class.</p> 
                          <p>The .container class provides a responsive fixed width container.</p>           
                        </div>
                        
                        </body>
                        </html>

Python

A popular programming language

Python can be used on a server to create web applications. It can be used alongside software to create workflows. It has a simple syntax similar to the English language.

Python Example:

                    import mysql.connector

                        mydb = mysql.connector.connect(
                          host="localhost",
                          user="myusername",
                          password="mypassword",
                          database="mydatabase"
                        )
                        
                        mycursor = mydb.cursor()
                        
                        sql = "SELECT \
                          users.name AS user, \
                          products.name AS favorite \
                          FROM users \
                          RIGHT JOIN products ON users.fav = products.id"
                        
                        mycursor.execute(sql)
                        
                        myresult = mycursor.fetchall()
                        
                        for x in myresult:
                          print(x)
